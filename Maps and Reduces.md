## Time Complexity
### Reduces
only work with associative operations
$T_{p} = O(\frac{T_{1}}{P} + T_{\infty}) = O(\frac{n}{P} + logn)$
### Maps
each el in data structure gets processed individually
$T_{p} = O(\frac{T_{1}}{P} + T_{\infty}) = O(\frac{n}{P} + 1) = O(\frac{n}{P})$
